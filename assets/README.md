# Assets

Store images, maps, GPX files, and export assets here.

## Suggested Layout

- `images/`
- `maps/`
- `gpx/`
- `icons/`

Do not add third-party media without clear usage rights.

## GPX

Seed GPX files in `gpx/` are research control-tracks unless explicitly marked as verified. They should not be treated as turn-by-turn navigation until checked against official route geometry and local road conditions.
