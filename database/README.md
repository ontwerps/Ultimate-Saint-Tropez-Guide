# Database

Structured source data lives here. Prefer CSV for early content and SQLite once relationships, filtering, and map generation become more important.

## Files

- `places.csv`: towns, villages, viewpoints, parking areas, and practical stops.
- `beaches.csv`: beach facts and suitability criteria.
- `restaurants.csv`: restaurant facts, tags, and source notes.
- `bike_routes.csv`: cycling routes by type.
- `hikes.csv`: hiking routes.
- `sources.csv`: reusable source references.
