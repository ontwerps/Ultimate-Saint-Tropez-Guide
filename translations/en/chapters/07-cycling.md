# Cycling

## Summary

This chapter separates trekking, road, gravel and e-bike routes. Safety, surface, traffic, elevation and stops matter more than distance or speed.

## Route Data

Routes live in `database/bike_routes.csv`; GPX files live in `assets/gpx/`. The first tracks are planning control tracks, not final navigation files:

- `prairies-port-grimaud-sainte-maxime-seed.gpx`
- `port-grimaud-frejus-v65-seed.gpx`
- `gassin-ramatuelle-pampelonne-loop-seed.gpx`

## Source Status

Port Grimaud - Sainte-Maxime uses current coastal-cycle context. Port Grimaud - Fréjus relates to V65 / Parcours Cyclable du Littoral context but needs official geometry checks. Gassin - Ramatuelle - Pampelonne is a road-cycling concept that needs local review of traffic, gradients, stops and safety.

## Route Profiles

Prairies de la Mer - Port Grimaud - Sainte-Maxime is the best first test: short, useful and revealing. Port Grimaud - Sainte-Maxime - Fréjus belongs to the wider V65 coastal context and needs official geometry before navigation. Gassin - Ramatuelle - Pampelonne is the sportier road loop and should be ridden early, never as a family route or wine-tasting add-on.

## By Bike Type

Trekking and e-bike suit the coast and practical errands. Road cycling belongs to early, controlled windows. Gravel is not yet mature in this guide because surfaces still need checking.

## Rules

Local GPX files must live under `assets/gpx/`; route records may only reference a local GPX file if it exists. External GPX URLs can be added later when licensing allows it.

## Personal Recommendation

The short Prairies de la Mer - Port Grimaud - Sainte-Maxime connection is the best first cycling test. Longer V65 and Gassin/Ramatuelle loops belong in a sporty plan only after surface, crossings, summer traffic and water stops are checked.
